Alaska Heat Pump Calculator API



This API provides data and calculations related to assessing the performance and economics of heat pumps installed in Alaska buildings. Endpoints are also provided for accessing the underlying Alaska community data, such as fuel and electric prices and weather data. Endpoints are available for doing general cash-flow economic analysis.

Two styles of documentation are available for the API:

  • Swagger style documentation: Here
  • Redoc style documentation: Here

Some example usage is available in this Jupyter Notebook using Python as the language to access the API.

In this Python module, there are additional notes about each of the keys of the JSON objects that hold the inputs to the available API endpoints. Notes on the returned JSON object keys are also available in the module.
